Для удобства использования у компонентов есть альясы:
- Block - Div
- Heading - H1, H2, H3, H4, H5 (без указания параметра size)
- Paragraph - P

Заголовки можно использовать только в правильной последовательности: нельзя ставить большой заголовок сразу 
после меньшего.

    распределены по отраслям.

    <Span color={Span.REFS.COLORS.MAJOR} isAccented> Сложно сказать, почему явные признаки победы институциализации призывают нас к новым свершениям, </Span>
    которые, в свою очередь, должны быть подвергнуты целой серии независимых исследований. 
    Лишь реплицированные с зарубежных источников, современные исследования объединены в целые кластеры себе
